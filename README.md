
# CloudStack Pricing Section

A responsive pricing section built using Tailwind CSS.  
This project demonstrates responsive layouts, utility-first styling, typography hierarchy, spacing systems, and interactive UI components using only Tailwind utility classes.

---

## Features

- Responsive pricing cards
- Mobile-first design
- Tailwind CSS utility classes only
- Highlighted "Most Popular" pricing plan
- Responsive grid layout
- Hover button interactions
- Consistent spacing and typography
- Hidden bonus features on smaller screens

---

## Technologies Used

- HTML5
- Tailwind CSS (CDN)

---

## Responsive Design

The pricing cards:
- Stack vertically on mobile devices
- Display in a 3-column grid on medium and larger screens using:
```html
md:grid-cols-3
```

## Reason for using CSS Grid
I used CSS Grid because it makes it easy to arrange the pricing cards into clean columns and rows. It also works well for responsive design since the cards stack on mobile and become multiple columns on larger screens.
